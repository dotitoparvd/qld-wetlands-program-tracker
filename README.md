# Queensland Wetlands Program Tracker

This repository serves as a living, public document to openly track the implementation, progress, and impacts of the **Queensland Wetlands Program**. 

## Purpose

The Queensland Wetlands Program is a key policy framework aimed at protecting, managing, and restoring wetlands across Queensland, Australia. This repository aims to:

- Document the program's core goals and strategies
- Track specific actions and milestones
- Provide a transparent record of progress for advocates, researchers, and community members
- Encourage collaboration and community involvement in wetland conservation efforts

## Structure

- **`policy-goals.md`**: Details the main objectives and key strategies of the Queensland Wetlands Program.
- **Issues**: Used to track specific actions, news items, and data reviews related to wetland protection.
- **Pull Requests**: Proposed updates and improvements to the documentation.

## Contributing

We welcome contributions from anyone interested in wetland conservation. Feel free to open issues to report new developments, suggest improvements, or add information about local projects.
